# Kubernetes and Ceph

A presentation of Kubernetes and Ceph using the [reveal.js](https://revealjs.com/) framework.

## Build

Build assets:

```sh
npm run build
```

## Run Speaker Mode

Start node-static server:

```sh
./node_modules/.bin/static
```

> Open [localhost:8080](http://127.0.0.1:8080/)
